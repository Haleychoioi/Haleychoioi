### 📚 주요 프로젝트

#### bookDam: 독서와 사람을 연결하는 커뮤니티 플랫폼

- **설명**: 기존 독서 플랫폼의 한계를 넘어, '책을 중심으로 깊이 있는 대화'가 가능한 커뮤니티 플랫폼입니다.
- **주요 기능**: 도서 검색, 커뮤니티 생성/참여, 마이페이지 독서 기록 관리, 독서 취향 분석 등
